네트워크 : 한 지점에서 원하는 다른 지점까지 의미있는 정보를 보다 정확하고 빠르게 상대방이 이해할 수 있도록 전송하는 것

거리순으로
PAN - LAN - MAN - WAN

데이터 전송방식
단방향
반이중
전이중

네트워크 토폴로지 : 네트워크 요소들(링크, 노드)을 물리적으로 연결해 놓은 것

1.계층형(트리구조)
2.버스형
3.성형(중앙장치)
4.링형
5.망형

회선교환 : 발신자와 수신자간에 회선을 독점하는 것

패킷교환 : 송신 측에서 모든 메세지를 일정한 크기의 패킷으로 분해해서 전송
	& 수신측엥서 이를 원래의 메세지로 조립

	[패킷] : 데이터 전송 단위

데이터그램 : 패킷 교환방식으로 동작하면서 IP 주소를 사용하는 인터넷
가상회선 : 회선 교환방식과 데이터그램 방식의 장점을 결합한 통신기술

메세지 교환 : 송신된 메세지를 중앙에서 축적하여 처리하는 방법 (전자우편)
