ls :  디렉토리 안 목록들 보여줌
pwd
type : 실행파일, 키워드, 내장 명령, 등 구분하는 명령
< : 표준입력
> : 표준출력
2> : 오류 표준출력
| : 한 명령이나 스크립트의 출력을 다른 명령의 입력으로 연결
& : 백그라운드 명령 실행
chmod : 권한부여
echo : 출력
= : 변수값 배정 -> 조회시에는 변수 이름 앞에 $ 붙이기
빈칸을 포함한 문자열을 변수에 배정하려면 ' 나 " 로 감싸야 함
" " 는 문자열의 일부를 변수의 값으로 치환이 가능 = 문자열 안에서 명령어로 사용 가능함.
( ) 를 사용하여 한 명령의 출력을 변수에 배정할 수 있다.

위치 매개변수
$0 : 스크립트 자체 이름
$1 : 첫 째 매개변수
$2 : 둘 째 매개변수...
$# : 전체 매개변수 갯수

read : 사용자 입력
조건분기는 if-elif-fi문 사용

grep : 파일을 한 줄씩 읽어서 주어진 패턴을 찾고, 패턴과 부합하는 부분이 있으면 그 행을 출력

메타문자(문자로 취급하려면 메타문자 앞에 \ 붙이기)
. : 하나의 와일드 카드 문자 하나
? : 생략해도됨을 지정
* : 앞의 항목이 0회 이상 반복될 수 있음을 지정하는 메타문자= 여러번 나타나도 됨
+ : 반복횟수가 1회 이상, 즉 앞에 적어도 한 번은 나타나야 함.
()를 이용하여 문자들을 하나의 그룹으로 묶을 수 있음
[] : 미리정의된 문자부류나 허용되는 문자들의 목록을 지정하는 용도로 쓰임

수량한정자
'문자'{}: 자신의 앞에 있는 항목이 몇 번 출현해야 하는지를 지정하는 역할

앵커와 단어 경계
^ : 그 다음의 패턴이 문자열 제일 처음 부분에서 부합해야 함
$ : 패턴이 문자열의 끝에서 부합해야 함
\b : 단어의 경계(빈칸, 시자과 끝)를 \b로 지정가능


사이버 보안 원칙들
(기밀성, 무결성, 가용성, 부인 방지, 인증)

공격작전[공격 수명 주기]
정찰 -> 초기 침투 -> 거점 확보 -> 권한 상승 -> 내부 정찰 -> 
횡적 이동 -> 존재 유지 -> 임무 완수
