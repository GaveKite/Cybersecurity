cut : 파일의 특정 부분을 추출하는 명령
file : 주어진 파일의 형식(유형)을 말해줌.
head : 파일의 처음 몇 행 또는 처음 몇 바이트를 출력
reg : Windows 레지스트리를 명령줄에서 조작
wevtutil : windows 환경에서 시스템 로그를 보거나 관리하는 데 사용하는 명령줄 유틸리티

Local System Information Collect[Git Bash]
uname -a : 운영체제 버전 정보
systeminfo : 시스템 하드웨어 및 관련 정보
ipconfig : 네트워크 인터페이스 정보
route print : 라우팅 테이블
arp -a : ARP
netstat -a : 네트워크 연결 정보
net share : 파일 시스템 구성
tasklist : 실행 중인 프로세스들

리눅스는 이름이 마침표로 시작하는 파일을 숨겨진 파일로 간주.
!! : 마지막으로 실행한 명령으로 대체

해시 : 그 파일의 내용을 고유하게 식별하는 값 == 다이제스트 값
(원본 글자가 하나만 달라도 해시의 전체값은 달라짐)
